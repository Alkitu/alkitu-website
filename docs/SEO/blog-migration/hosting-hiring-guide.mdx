# Todo lo que debes saber antes de contratar tu Hosting

Si lo que quieres tener tu propio sitio web uno de los requisitos primordiales contar con tu propio dominio y un proveedor de Hosting o alojamiento. En esta publicación te daremos unos tips primordiales que debes tener en cuenta antes de contratar tu proveedor de servicios de hosting.

## ¿Qué es un Hosting o alojamiento?

Un Hosting o alojamiento web es un servicio auspiciado por una empresa mediante el cual se aloja todo el contenido de tu sitio web para que se pueda visitar e interactuar en linea. Cuando quieres acceder a cierta información en tu ordenador esta se encuentra alojada en la memoria interna del mismo, ocurre de igual manera en los sitios web, estos deben están alojados en un servidor de alojamiento o hosting para estar en linea.

### ¿Cómo funciona el servicio de Hosting?

Una vez se cuenta con el servicio de Hosting web, puedes alojar los archivos de tu sitio web en el mismo para posteriormente acceder a ellos en cualquier momento haciendo uso de tu dominio web (si quieres conocer más información acerca de dominios echa un vistazo a nuestra publicación).

Una vez escrito el nombre del dominio en la barra buscadora del navegador. Una vez accedes al dominio, el navegador irá solicitando progresivamente los datos de tu web al servidor de tu hosting y mediante los DNS se irán ubicando cada archivo correspondiente. De esta manera, estos irán siendo mostrados a través del navegador web.

### Tipos de Hosting

**Hosting Compartido**

Es el tipo de Hosting más utilizado y consiste en alojar las cuentas de hosting de diferentes usuarios en un mismo servidor físico. Esto implica que algunos recursos como la memoria RAM y la CPU son compartidos.

**Hosting VPS**

El Hosting VPS (Virtual Private Server) es un servidor privado virtual en el cual no se comparten los recursos. A pesar de estar en un mismo servidor físico estos VPS cuentan con un espacio y cantidad de recursos asignados y garantizados solo para ellos.

**Hosting elástico**

El hosting elástico reúne las características positivas de un hosting compartido y un hosting VPS. Con esto queremos decir que se comparten recursos físicos con otros usuarios pero también tenemos recursos garantizados como el VPS.

**Hosting Cloud**

El hosting cloud o servidor en la nube es un tipo de servidor que funciona desde una infraestructura compuesta de varios servidores que trabajan en conjunto y de esta manera el servicio se distribuye en diferentes equipos conectados a una misma red, formando la nube.

Estos servidores son usados en línea general por empresas con demanda de recursos inestables, la cuales suelen tener variaciones de necesidades y deben poder solventarlo en brevedad. La nube ofrece una gran cantidad de recursos disponibles y garantía antes esas variaciones de demanda.

**Servidor dedicado**

Un servidor dedicado es un tipo de hosting web en el que se ofrece exclusivamente un equipo físico completo para cada cliente. De esta manera no se comparte el servidor con ningún usuario así se tienen todos los recursos garantizados.

Es importante tener en cuenta cuales podrían llegar a ser las demandas de recursos de tu sitio web antes de contratar el servidor. Así podrás satisfacerlas y optimizar el funcionamiento de tu sitio web en internet.
