## Cómo ejecutar

### Opción 1: Supabase SQL Editor (recomendado)
1. Ir a [app.supabase.com](https://app.supabase.com)
2. Seleccionar el proyecto Alkitu
3. Ir a **SQL Editor**
4. Copiar y pegar el contenido completo de `20260320000001_restructure_categories.sql`
5. Hacer clic en **Run**
6. Verificar que los mensajes NOTICE confirmen 6 categorías y las asociaciones correctas

### Opción 2: Supabase CLI
```bash
supabase db push
```

### Opción 3: psql directo
```bash
psql -h db.YOUR_PROJECT_REF.supabase.co -p 6543 -U postgres -d postgres -f supabase/migrations/projects/20260320000001_restructure_categories.sql
```

## Verificación post-migración

Ejecutar en SQL Editor:
```sql
-- Ver categorías actuales
SELECT name_en, name_es, slug FROM categories ORDER BY name_en;

-- Ver asociaciones por proyecto
SELECT p.title_en, string_agg(c.name_en, ', ' ORDER BY c.name_en) as categories
FROM projects p
JOIN project_categories pc ON p.id = pc.project_id
JOIN categories c ON pc.category_id = c.id
GROUP BY p.title_en
ORDER BY p.display_order;
```
